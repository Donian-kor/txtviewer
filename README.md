# TXT Viewer (Python, Tkinter)

간단한 TXT 파일 뷰어 프로그램입니다.  
다크 모드 UI와 다양한 편의 기능을 제공합니다.

## ✨ Features
- Dark Mode UI
- 좌측: TXT 파일 목록 / 우측: 텍스트 뷰어
- 자동 인코딩 감지 (UTF-8, CP949 등)
- 상태 표시줄: 라인 수, 파일 크기, 인코딩 정보
- 폰트/크기/배경/글자색 변경 가능
- 설정 저장/불러오기 (`settings.json`)
- 마지막 폴더 및 마지막 TXT 파일 자동 불러오기
- 옵션 GUI 창: 한 번만 열리고 항상 메인창 오른쪽에 위치

## 🖥️ 사용법
1. `파일 → 폴더 열기` 메뉴에서 TXT 파일이 있는 폴더 선택
2. 좌측 목록에서 파일 클릭 → 우측 뷰어에 내용 표시
3. `옵션 → 폰트/배경/색상 설정`에서 스타일 변경 후 `적용` 버튼 클릭
4. 프로그램 종료 후 다시 실행해도 마지막 설정/폴더/파일 자동 불러오기

## ⚙️ 요구 라이브러리

- tkinter
- chardet
